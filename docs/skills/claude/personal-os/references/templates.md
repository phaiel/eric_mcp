# Output templates

## Find the thread

```
DECISION: [one sentence, or "No captured decision yet"]
STILL OPEN:
- [≤3]
CONTEXT: [≤3 sentences]
→ [optional next action]
```

## Today's focus

```
TODAY (pick up to 3):
1. [action — project — energy]
BLOCKING YOU: [0–2]
IGNORE FOR NOW: [one line]
```

## Decision memo

```
OPTIONS: [≤3]
ASSUMPTIONS:
RISKS:
RECOMMENDATION: [one sentence + confidence]
→ Capture as decision?
```

## Execution audit

```
STALE (>2 weeks, ≤5):
OPEN DECISIONS (≤5):
FOLLOW-UPS YOU OWE:
→ Pick one to unblock?
```

## Capture confirm

```
Captured: "[title]"
→ Filed under: [Project] as [Action/Decision/Inbox]
```

## Energy fit

| Stated energy | Prefer |
|---------------|--------|
| High | Deep work, decisions |
| Medium | Standard tasks |
| Low | Admin |
| Zombie | One <15-min win |

"I'm fried" / "20 minutes" → filter accordingly.
