# Google Workspace MCP (when connected)

Universal Search — hosted by Google. Typically one tool: `search_corpus`.

## Use for

- **Find the thread** across email, Drive, Calendar, Chat
- "Anything about X in my workspace"
- Pre-meeting context spanning products

## Do not use for

- Creating calendar events, sending mail, editing Drive files → not available on this connector (add product MCP later)
- Replacing Notion commitments layer

## Query tips

- One clear natural-language query
- Mention product hints if needed ("in email", "on my calendar")
- Summarize results; don't dump raw JSON

## Not connected

If `search_corpus` is absent, say one line and use Notion/Hevy only.
