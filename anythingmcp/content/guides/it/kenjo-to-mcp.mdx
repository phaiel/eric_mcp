## Kenjo HR con Agenti AI

Kenjo è una piattaforma HR tedesca. Con AnythingMCP, colleghi la sua API a qualsiasi agente AI.

### Configurazione

Importa l'adattatore **Kenjo**, inserisci la API key e collega il tuo agente AI.
